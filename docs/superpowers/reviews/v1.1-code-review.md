## 三、发现的问题清单

### 🔴 Critical (必须修复)

**C-1: merge-entities 路径穿越漏洞**
- **文件**: `api/routers/maintenance.py` L383-L384
- **问题**: 直接拼接用户输入构造文件路径，未使用 `_safe_path()`

```python
keep_file = WIKI_PATH / f"{keep_page}.md"      # ❌ 无保护
remove_file = WIKI_PATH / f"{remove_page}.md"   # ❌ 无保护
```

- **风险**: 用户传 `keep_page=../config` 可读写 Wiki 目录外的文件
- **修复**: 改用 `_safe_path(WIKI_PATH, keep_page)` / `_safe_path(WIKI_PATH, remove_page)`

---

### 🟡 High (建议修复)

**H-1: `_build_graph_data` 无 async 操作**
- **文件**: `api/routers/graph.py` L38
- **问题**: 函数声明为 `async def` 但内部全是同步 IO 操作 (读文件、正则匹配)，没有 `await`
- **影响**: 功能正常，但会误导后续维护者 (以为有异步优化空间)；调用方用 `await` 可正常运行
- **修复**: 改为 `def`，调用方去掉 `await`

**H-2: search.py 和 maintenance.py 中 `sys.path.insert` 重复调用**
- **文件**: `api/routers/search.py` L141, `api/routers/maintenance.py` L60/L248/L478
- **问题**: 每次请求都往 `sys.path` 追加 SCRIPTS_DIR，高频调用会导致列表持续增长
- **修复**: 改为仅在路径不存在时才 `insert`

**H-3: pdf.py 上传错误不记录日志**
- **文件**: `api/routers/pdf.py` L34-L36
- **问题**: 文件类型错误、文件过大只抛 HTTPException，未调用 LogService
- **修复**: 在 raise 前用 `get_db_ctx()` 写入 system_log

---

### 🔵 Medium (建议改进)

**M-1: ingest.py 中 `import sys as _sys` 冗余**
- **文件**: `api/routers/ingest.py` L50 (`import sys as _sys`) + L6 (`import sys`)
- **问题**: 同一文件内定义了两次 `sys` 的导入，`_sys` 别名不必要
- **修复**: 删除 `import sys as _sys`，统一使用 `sys`

**M-2: maintenance.py 同样存在 `import sys as _sys` 冗余**
- **文件**: `api/routers/maintenance.py` L470
- **问题**: `import sys as _sys` 但顶部已 `import sys`
- **修复**: 删除别名，统一使用 `sys`

**M-3: datetime 时区不一致**
- **文件**: `api/routers/pdf.py` L131 `datetime.now(timezone.utc)` vs L75 `datetime.now()` (naive)
- **问题**: 同一文件中时区处理不统一
- **修复**: 统一使用 `datetime.now()` 或统一带时区

**M-4: config.py update_config 缺响应模型**
- **文件**: `api/routers/config.py` L73
- **问题**: 返回裸 `dict`，API 文档中看不到响应结构
- **修复**: 添加 `response_model`

**M-5: uploadPDF 缺少 401 重定向逻辑**
- **文件**: `web/src/services/api.ts` L210-L226
- **问题**: `uploadPDF` 用 fetch 直连，不像 `request()` 有 401 自动跳转登录页
- **修复**: 加 401 检测或统一走 `request()` 包装

---

### 🟢 Low (可选优化)

**L-1: agent_g.py retry 中 import time 在函数内部**
- `scripts/agent_g.py` L41 — 移到文件顶部更规范

**L-2: .gitignore 中 `*.log` 规则粗粒度**
- 覆盖了 `batch.log` 但可能误伤 `changelog` — 如需追踪 debug log，细化规则为 `/logs/*.log`
